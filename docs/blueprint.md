# **App Name**: Youth Opportunities Hub

## Core Features:

- Opportunity Listings: Display a curated list of extracurricular opportunities, filtered by type, subject, price, audience, format, age, and deadline.
- Bookmark Opportunities: Allow students to save opportunities, for reviewing later.
- Submit New Opportunity: Enable students to submit new opportunities for admin review and approval, including fields for title, type, description, category, age range, price, audience, format, deadline, and registration link.
- Admin Dashboard: Implement a dedicated admin dashboard for viewing, reviewing, accepting, rejecting, editing, and deleting submitted opportunities, including direct opportunity addition.
- Opportunity Detail Page: Design dedicated opportunity detail pages, accessible to logged-in students, presenting comprehensive details with a save/bookmark option and an edit option for admins.

## Style Guidelines:

- Primary color: Midnight Blue (#191970), offering a sense of professionalism and depth.
- Background color: Light Gray (#D3D3D3), for a modern, uncluttered appearance.
- Accent color: Electric Blue (#7DF9FF) for highlights and interactive elements, adding a contemporary and energetic touch.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, providing a tech-forward feel, paired with 'Inter' (sans-serif) for body text, ensuring readability.
- Use a set of consistent, modern icons to represent opportunity types and filter options; icons should be in the accent color.
- Implement a left sidebar for filters on the dashboard, ensuring it's always visible on desktop for easy navigation; prioritize a card-based design for opportunity listings.
- Introduce subtle animations on button hovers, modal transitions, and form feedback to enhance user engagement and provide clear visual cues.
